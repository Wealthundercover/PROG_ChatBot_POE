# Cybersecurity Awareness Chatbot - POE Part 1

## Project Overview
This is a C# Console Application designed to educate users on essential cybersecurity practices. The bot provides interactive guidance on identifying phishing attempts, creating strong passwords, and maintaining safe browsing habits.

## Key Features
* **Interactive Persona:** Personalized user interaction with a dedicated greeting system.
* **Multimedia Integration:** * **Audio:** Plays a custom voice greeting upon startup using `System.Media`.
    * **Visuals:** Displays a stylized ASCII Art header for a professional interface.
* **Security Modules:**
    * **Phishing Detection:** Advice on spotting suspicious emails and links.
    * **Password Security:** Tips for creating uncrackable credentials.
    * **Safe Browsing:** Best practices for navigating the web securely.
* **Robust Architecture:** Built using a decoupled design with a dedicated `ChatbotEngine` class for logic processing.

## Technical Specifications
* **Language:** C#
* **Framework:** .NET 9.0+
* **Version Control:** Managed via GitHub with automated CI/CD.
